# Lagos 2023 Presidential Vote Race Animation

Animated bar race showing APC, LP, and PDP vote counts for Lagos State, Nigeria 2023 Presidential Election.

## Demo
![Animation Demo](demo.gif) 

## Data
Source: INEC Nigeria 2023 Presidential Results

## How to Run
1. Clone repo: `git clone https://github.com/shakexess/lagos-2023-vote-race.git`
2. Install: `pip install -r requirements.txt`
3. Open `inec_vote_race.ipynb` in Jupyter
4. Run all cells

## Tech Stack
Python, Matplotlib, Pandas, Jupyter
